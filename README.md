# Morgan Shop

Furniture shop. 

- Shopping bucket logic
- Dynamic routing
- Scroll button logic
- Skeletons while loading images

[Demo version](https://seventrio.github.io/morgan-shop/)

### Used dependencies and technologies:
- React
- Redux
- Redux Thunk
- Reselect
- React Router
- React Scroll
- React Portal
- React Transition Group
- Scss
- CSS Flexbox
- CSS Grid
